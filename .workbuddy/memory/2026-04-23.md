### Bug 修复 (17:25)
- **页面卡在"加载中"问题**：根本原因是 Google Fonts (fonts.googleapis.com) 在国内被墙，CSS 请求阻塞了整个页面渲染
- 修复：移除 Google Fonts 依赖，改用系统字体 + bootcdn 国内 CDN 加载 Font Awesome（带 fallback）
- 修复：Flask debug=True 后台运行会出问题，改为 debug=False + threaded=True
- 修复：send_from_directory 相对路径可能错误，改为 os.path.abspath 绝对路径
- 添加 import os 到 app.py 顶部
- **重要提醒**：必须通过 http://localhost:5000/ 访问，不能用 file:// 协议打开本地 HTML
